## :black_nib: Rodando o projeto

Clone o projeto e acesse sua pasta.

```bash
$ git clone https://github.com/fabiomoura-m/ignite-feed.git
$ cd ignite-feed
```

Siga os passos:

```bash
# Install the dependencies
$ yarn or npm i
# Start the project
$ yarn dev or npm run dev
```
